# Numeros-Decrecientes
 Esté código pertenece a un ejercicio realizado en el curso profesional Javascript en la Asociación Española de Programadores Informáticos (AEPI).

## Descripción ⌨️
Crea un script donde el usuario introduzca 3 valores númericos, y el programa los ordene de manera decreciente.

El Objetivo principal de este ejercicio es el uso de funciones que reciben parámetros.
